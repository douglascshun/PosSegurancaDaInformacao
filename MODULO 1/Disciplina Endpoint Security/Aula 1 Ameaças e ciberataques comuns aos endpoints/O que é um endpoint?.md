# O que é um endpoint?

É o ponto de extremidade, ou seja, equipamentos na ponta que se conectam a uma rede.

- `Para eu considerar um endpoint preciso que o equipamento tenha um endereço IP`

### Existem dois tipos de endpoint de comunicação

`Endpoint de comunicação` se trata do endereço de web (URL), aquele texto que você digita para encontrar algum site, página ou serviço.

Sua definição está entre a conexão de uma ponta A (navegador do usuário) com uma ponta B (web server do servidor).

`Endpoint de segurança`faz parte do conjunto de soluções ara mitigar qualquer ação maliciosa do web service. Sua abordagem atua protegendo a atividade na borda da estrutura de rede.

Refere-se à proteção do acesso de dispositivos (computadores notebooks, smartphones ou tables) conectados na rede corporativa.

A segurança de endpoint muitas vezes é confundida com um antivírus ou um firewall. Embora todos eles sejam ferramentas na proteção digital, cada um deles tem uma tarefa diferente.

Em um sistema de cibersegurança, endpoint security é uma medida de estratégia para impedir que os dados do seu negócio ou de seus clientes, sejam alcançados por hackers pelas portas de entrada, como computadores e dispositivos móveis.
