## O que é hash?

Algoritmos, baseados em funções matemáticas, que mapeiam dados de comprimento variável para dados de comprimento fixo.

O algoritmo de hash cria a partir de uma cadeia de caracteres de qualquer tamanho uma string de tamanho fixo.

Hash são caminhos de mão única (funções matemáticas) que são utilizadas para se criar códigos. Qualquer cifra criada com algoritmo hash se torna "impossível" matematicamente de se recuperar.

# Características dos algoritmos de hash

Não possuir chave, ou seja, gerar o hash não envolve nenhum segredo que deva ser compartilhado. Isso representa força e problema;

Tamanho único, logo não importa o quão grande é a informação original, o hash será sempre de um mesmo tamanho.

Exemplo: versão MD5 gera código de 128 bits de tamanho, mesmo que seja um arquivo de 1 único Byte (8 bits) ou um de 1 GB.

Hash é um termo da língua inglesa que pode ser traduzida com vários significados, como `picar, coisa perfeita, misturar e confusão`.

# Valores retornados por uma função hash

- `Valores hash`
- `Códigos hash`
- `Somas hash (hash sums)`
- `Checksums`


# Principais algoritmos de hash

- `MD5`
- `RIPMED`
- `SHA-1 (160)`
- `WIRLPOOL`
- `SHA-2 (224, 256, 284 e 512)`
- `OPEN SSL com SHA`
