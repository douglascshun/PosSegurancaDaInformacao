## Proteção e criptografia de dados

O serviços Defender para Endpoint utiliza tecnologias de proteção de dados de última geração baseadas em Microsoft infraestrutura do Azure.

A criptografia é uma das mais críticas e inclui criptografia de dados em repouso, criptografia em voo e gerenciamento de chaves Key Vault.. Em todos os cenários, os dados são  criptografados usando a criptografia AES de 256 bits no mínimo.
