[[Aula 2 Infraestrutura de Segurança Para Nuvem]]

Os data centers tem papel importante na Nuvem!
	São ambientes especialmente projetados para alocação de componentes eletrônicos de alto desempenho, como servidores, roteadores, redes de área de armazenamento ... cuja principal função consiste em fornecer um ambiente altamente controlado para processamento e armazenamento de dados.
