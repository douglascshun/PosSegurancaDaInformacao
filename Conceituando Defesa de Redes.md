# O que é rede?

![[Exemplo de rede.png]]Uma rede são conexões entre dispositivos que compartilham recursos e/ou arquivos, no exemplo da imagem se é possível ver uma rede que tem o recurso impressora compartilhada entre os dispositivos locais, a rede é distribuída por meio de cabos e uma rede wi-fi, além disso também se é possível analisar que essa rede se conecta em um modem que dá acesso a internet.
