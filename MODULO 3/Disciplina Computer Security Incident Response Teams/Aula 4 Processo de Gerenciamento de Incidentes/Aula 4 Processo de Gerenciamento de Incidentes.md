[[Computer Security Incident Response Teams]]

Proposito do processo de gerenciamento de incidentes
	A estrutura serve para planejar uma capacidade de gerenciamento de incidentes e para fornecer uma metodologia para avaliar seu trabalho.
		Serve também para entender como os CSIRTs se encaixam no esquema geral de gerenciamento de incidentes e na estrutura de suas organizações principais ou de hospedagem.

O modelo fornece processos para descrever:
	- Principais funções
	- Atividades necessárias

O modelo pode ser usado por uma organização para:
	- Planejar uma nova capacidade
	- Comparar sua capacidade atual
	- Fornecer um caminho para melhorar e expandir a capacidade
