[[Aula 5 Gerenciamento de Incidentes]]

Os `incidentes de segurança`, são eventos que podem indicar que os sistemas ou dados de uma organização foram comprometidos ou que as medidas implementadas para protegê- los falharam.
	Um `evento de segurança` é qualquer coisa que tenha importância para o hardware ou software do sistema, e um `incidente` é um evento que interrompe as operações normais.
